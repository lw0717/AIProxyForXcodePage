AIProxy for Xcode

## AIProxy

AIProxy for Xcode (hereinafter referred to as "AIProxy") is an macOS App. AIProxy is a lightweight local HTTP server that provides a proxy service for AI large language models that support the OpenAI-style API (such as Deepseek, Kimi, etc.) for use with Xcode.

## How to use

1. Download the latest version of AIProxy.

2. Go to Xcode App - Setting - Interlligence

3. Click "Add a Model Provider..."

4. Add http://127.0.0.1:7170/ in "Internet Hosted"

5. Select the model you added 

6. Start using the corresponding large language model in Xcode.
